🛠️ 2. Build a Mini Tool

    Goal: Build something small (e.g. a weather app or email sender).

##################################################
✅ Step-by-Step:

    🟢 Step 1: Pick the Tool Type

        Examples:
            CLI tool (runs in terminal)
            Web app (runs in browser)


    🟢 Step 2: Define What It Does

        Be clear and plan what function to build.
        Example: “This tool should get weather by city name.”


    🟢 Step 3: Break Into Steps

        Example: Weather App
            1. Ask user for a city name
            2. Call a weather API
            3. Show the weather


    🟢 Step 4: Pick Your Language

        Example programming languages:
            Python (great for CLI tools)
            JavaScript (good for web apps)


    🟢 Step 5: Use an API

        Find a simple API like:
        https://openweathermap.org/api
        Sign up and get an API key if needed.


    🟢 Step 6: Write the Code

        Start small. One step at a time:
            - Input
            - API call
            - Output


    🟢 Step 7: Test Often

        Run the tool often while building.
        Fix problems bugs early.


    🟢 Step 8: Organize Code

        Separate:
            - Input code
            - API code
            - Output/Display

##################################################

💡 Tips:

    Keep it simple.
    Use comments ✍️ to explain the code. 
    Try using libraries like:
        Python: requests, argparse
        JavaScript: fetch, axios
